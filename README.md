# Zscript
It does nothing

## Installation
Install Zscript with pip
```bash
  pip install -r requirements.txt
```

## Guide
```main.zscript``` Main script file

```override.zmodule``` Python code override

```basic_module.zscript``` Basic module

```[End]``` Clear output

```#0.1;``` Add 0.1 second delay

#### Compile
```bash
  python3 compiler.py
```

#### Run
```bash
  python3 output.py
```
